🤖 Smart Health Assistant Bot
📌 Project Overview

The Smart Health Assistant Bot is an AI-based chatbot designed to provide basic health-related suggestions and information. It interacts with users through text input and responds based on predefined data and simple Natural Language Processing (NLP) techniques.

This project is developed as part of an Artificial Intelligence course to demonstrate practical implementation of chatbot systems.

🎯 Objectives
To build a simple AI chatbot
To understand basic NLP concepts
To simulate human-like conversation
To apply AI in healthcare assistance
🚀 Features
💬 User-friendly chatbot interaction
🧠 Basic NLP-based response system
📊 Dataset-driven answers
⚡ Fast and simple execution
🛠️ Technologies Used
Python
Natural Language Processing (NLP)
CSV Dataset
Basic Machine Learning Concepts
📂 Project Structure
AI_Project/

▶️ How to Run the Project

Install Python on your system
Open terminal or command prompt
Navigate to the project folder
Run the following command:
python main.py

User: "I have a headache"
Bot: "You should take rest and drink plenty of water."
